# Mina zkApp: Sliced Contracts

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```

## How to run lightnode

```sh
zk lightnet start
```
